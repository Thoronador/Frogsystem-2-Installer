
----- Frogsystem 2.alix7! ------------------------------------------------------------------
----- Datei-Operationen manuell ausführen --------------------------------------------------

Bei einem Update von einer älteren Version bitte die einzelnen Updates hintereinander
ausführen, wenn keine speziellen Anweisungen existieren.

----- Installation -------------------------------------------------------------------------

   1) Haupt-Dateien kopieren

      => Alle Dateien aus dem Ordner /fs2installer/copy/source in das Zielverzeichnis der
         Installation kopieren.

      => Alle Dateien aus dem Ordner /fs2installer/copy/data in das Zielverzeichnis der
         Installation kopieren. Bereits vorhandene Ordner/Dateien überschreiben.


   2) Zugriffsrechte festlegen

      => Der Webserver benötigt Schreib-Zugriff für folgende Dateien und Ordner:

            /styles (sowie alle Unterordner und Dateien)
            /media  (sowie alle Unterordner und Dateien)
            /upload 
            
      !! Weiterführende Informationen gibt es im Frogsystem 2-Wiki unter:
         https://github.com/mrgrain/Frogsystem-2/wiki/filesystem-permissions


   3) Datenbank-Verbindung einrichten

      => Die Datei /config/env-example.cfg.php aus dem Zielverzeichnis umbennen in
         "env.cfg.php" und die Verbindungsdaten zur Datenbank eingegeben.

      !! Diese Datei sollte vor unbefugten Zugriffen geschützt werden und nicht von anderen
         Benutzern außer dem Webserver und dir selbst lesbar sein. Siehe dazu ebenfalls:
         https://github.com/mrgrain/Frogsystem-2/wiki/filesystem-permissions


----- Update von 2.alix6(b) auf 2.alix7 ----------------------------------------------------

   1) Veraltete Dateien löschen

      => Diese Ordner (inkl. Dateien) aus dem Zielverzeichnis der Installation sind veraltet
         und sollten vorher gelöscht werden:
            
            /admin/
            /classes/
            /feeds/
            /includes/
            /libs/
            /resources/
            /styles/default/
            /imageviewer.php
            /login.inc.php
            /config/db_connection.php
            
   2) Neue/Geänderte Dateien kopieren

      => Folgende Dateien aus dem Ordner /fs2installer/copy/source in das Zielverzeichnis der
         Installation kopieren:
         
            /admin/
            /libs/
            /applets/
            /assets/
            /config/
            /data/
            /lang/
            /media/stopwords/
            /index.php

   3) Dateien verschieben

      => Diese Ordner (inkl. Dateien) im Zielverzeichnis müssen an einen anderen Ort verschoben
         bzw. umbenannt werden. ("images" sollte danach leer sein und kann gelöscht werden)

            /images/cat/            =>   /media/cat/
            /images/downloads/      =>   /media/downloads/
            /images/partner/        =>   /media/partner/
            /images/press/          =>   /media/press/
            /images/screenshots/    =>   /media/gallery/
            /images/shop/           =>   /media/shop
            /images/smilies/        =>   /media/smilies
            /images/wallpaper/      =>   /media/wallpaper

   4) Zugriffsrechte festlegen

      => Der Webserver benötigt Schreib-Zugriff für folgende Dateien und Ordner:

            /styles (sowie alle Unterordner und Dateien)
            /media  (sowie alle Unterordner und Dateien)
            /upload

----- Update von 2.alix5 auf 2.alix6b ------------------------------------------------------

   1) Veraltete Dateien löschen

      => Diese Ordner (inkl. Dateien) aus dem Zielverzeichnis der Installation sind veraltet
         und sollten vorher gelöscht werden:

            /styles/default
            /resources
            /upload


   2) Haupt-Dateien kopieren

      => Alle Dateien aus dem Ordner /fs2installer/copy/source in das Zielverzeichnis der
         Installation kopieren. Bereits vorhandene Ordner und Dateien überschreiben.

      => Dateien aus dem Ordner /fs2installer/copy/data in das Zielverzeichnis der
         Installation kopieren. Bereits vorhandene Dateien überprüfen und NUR BEI BEDARF
         überschreiben.

      !! Schau dir die Änderungen aus /fs2installer/copy/data im Vergleich zu den aktuellen
         Dateien deiner Installation an. Vielleicht ist etwas Wichtiges oder Interessantes
         für dein Frogsystem 2 dabei.


   3) Inhaltsbilder verschieben

      => Die Inhalsbilder müssen an einen neuen Speicherort verschoben werden. Bitte
         verschiebe oder kopiere alle Dateien aus

            /images/content      in den Ordner     /upload


   4) Zugriffsrechte festlegen

      => Der Webserver benötigt Schreib-Zugriff für folgende Dateien und Ordner:

            /styles (sowie alle Unterordner und Dateien)
            /upload
            /media/content
            /media/group-images
            /media/user-images
            /images/cat
            /images/downloads
            /images/partner
            /images/press
            /images/screenshots
            /images/shop
            /images/smilies
            /images/wallpaper

      !! Weiterführende Informationen gibt es im Frogsystem 2-Wiki unter:
         http://wiki.frogsystem.de/filesystem-permissions


   5) Datenbank-Verbindung einrichten

      => In der Datei /fs2installer/copy/db_connection.php die Verbindungsdaten zur
         Datenbank eingegeben.

      => Kopiere diese Datei den Ordner /config im Zielverzeichnis. Die vorhandene Datei
         muss überschrieben werden.

      !! Diese Datei sollte vor unbefugten Zugriffen geschützt werden und nicht von anderen
         Benutzern außer dem Webserver und dir selbst lesbar sein. Siehe dazu ebenfalls:
         http://wiki.frogsystem.de/filesystem-permissions


   6) Alte Dateien aufräumen

      => Wir empfehlen NACH der Installation die folgenen veralteten und nicht mehr
         verwendeten Dateien und Ordner vom Webserver bzw. Webspace zu löschen:
